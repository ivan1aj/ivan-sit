# شرح التحسينات والتعديلات على موقع إيفان متعدد اللغات

## التحسينات الرئيسية

1. **تصميم عصري واحترافي**
   - تم تحديث التصميم ليكون أكثر احترافية وحداثة
   - استخدام ألوان واضحة وخطوط منظمة تناسب مواقع الشركات
   - تصميم متجاوب يعمل على جميع الأجهزة (الحاسوب، الجوال، اللوحي)

2. **صور احترافية وواقعية**
   - تم استبدال جميع الصور بصور احترافية تعبر عن مجال العمل
   - صور تعكس طبيعة العمل في تصميم المواقع وواجهات المستخدم
   - تناسق بصري بين جميع الصور في الموقع

3. **نظام متعدد اللغات**
   - دعم ثلاث لغات: العربية، الإنجليزية، الهولندية
   - تحديد اللغة الافتراضية تلقائياً حسب لغة متصفح المستخدم
   - إمكانية التبديل اليدوي بين اللغات بسهولة
   - تغيير اتجاه الصفحة تلقائياً (RTL للعربية، LTR للإنجليزية والهولندية)

4. **إضافة البريد الإلكتروني**
   - إضافة البريد الإلكتروني info@iivan.org في قسم التواصل
   - إضافة البريد في تذييل الموقع للوصول السريع
   - ربط البريد بخاصية mailto: لفتح تطبيق البريد تلقائياً

5. **زر طلب الخدمة**
   - إضافة زر ثابت لطلب الخدمة في أسفل يمين/يسار الشاشة
   - نافذة منبثقة لتعبئة نموذج طلب الخدمة
   - خيارات متعددة للخدمات المطلوبة

## هيكل الملفات

- **index.html**: الصفحة الرئيسية للموقع
- **assets/css/style.css**: ملف الأنماط والتصميم
- **assets/js/translations.js**: ملف الترجمات متعدد اللغات
- **assets/js/main.js**: ملف جافاسكريبت الرئيسي للوظائف التفاعلية
- **assets/js/gallery-data.js**: بيانات معرض المشاريع
- **assets/images/**: مجلد الصور الاحترافية

## كيفية النشر

1. **استخدام الرابط الدائم**
   - يمكنك استخدام الرابط الدائم مباشرة: https://nrztykro.manus.space
   
2. **الربط مع النطاق الخاص iivan.org**
   - قم بإضافة سجل CNAME في إعدادات DNS للنطاق يشير إلى الرابط الدائم
   - أو قم برفع محتويات الملف المضغوط إلى استضافة النطاق الخاص بك

3. **تعديل البريد الإلكتروني (إذا لزم الأمر)**
   - إذا كنت ترغب في تغيير البريد الإلكتروني، يمكنك تعديله في ملف index.html

## ملاحظات إضافية

- تم اختبار الموقع على مختلف المتصفحات والأجهزة للتأكد من التوافق
- النظام متعدد اللغات يدعم إضافة لغات جديدة بسهولة في المستقبل
- تم تحسين سرعة تحميل الموقع من خلال تحسين حجم الصور وتنظيم الكود
